# MVP Scope

**In-scope (MVP):**
- React Native mobile app (user + agent roles in one app).
- Backend APIs (Node/Express).
- PostgreSQL persistence.
- Location-based agent discovery (within radius).
- OTP-based verification (time-limited, single-use).
- One active transaction per user.
- Basic ratings for agents.
- Simple admin approval toggle for agents.

**Out-of-scope (MVP):**
- Full KYC automation.
- Blockchain.
- Offline mode.
- AI recommendations/demand prediction.
